Usage Example
How to use the Pagination component in your Django views: 
from djust.components.ui import Pagination

# In your view
pagination = Pagination(
    current_page=5,
    total_pages=20,
    size="md",
    show_first_last=True,
    max_visible_pages=5
)

# In your template
{{ pagination.render }}
# or simply
{{ pagination }}
