## What This PR Does

Completes the round-trip for `@debounce` and `@throttle` decorators:
1. **Phase 1** (PR #39): Python decorators inject metadata into HTML
2. **Phase 2** (this PR): JavaScript reads metadata and applies timing control
3. **Result**: Zero manual JavaScript required for users

### Before (Manual JavaScript)
```javascript
// 889 lines of manual debouncing code
let debounceTimer = null;
searchInput.addEventListener('input', (e) => {
  clearTimeout(debounceTimer);
  debounceTimer = setTimeout(() => {
    sendEvent('search', { query: e.target.value });
  }, 500);
});
```

### After (Pure Python)
```python
# Just add a decorator!
@debounce(wait=0.5, max_wait=2.0)
def search(self, query: str = "", **kwargs):
    self.results = Product.objects.filter(name__icontains=query)
```

1. **Global State Management** (client.js:33-39)
   - `debounceTimers` Map for per-handler debounce state
   - `throttleState` Map for per-handler throttle state

**debounceEvent()** (lines 214-256):
```javascript
debounceEvent(eventName, eventData, config) {
    const { wait, max_wait } = config;
    const now = Date.now();

    // Get or create state
    let state = this.debounceTimers.get(eventName);
    if (!state) {
        state = { timerId: null, firstCallTime: now };
        this.debounceTimers.set(eventName, state);
    }

    // Clear existing timer
    if (state.timerId) {
        clearTimeout(state.timerId);
    }

    // Check if we've exceeded max_wait
    if (max_wait && (now - state.firstCallTime) >= (max_wait * 1000)) {
        this.sendEventImmediate(eventName, eventData);
        this.debounceTimers.delete(eventName);
        this.debug('debounce', `Force executing ${eventName} (max_wait exceeded)`);
        return;
    }

    // Set new timer
    state.timerId = setTimeout(() => {
        this.sendEventImmediate(eventName, eventData);
        this.debounceTimers.delete(eventName);
        this.debug('debounce', `Executing ${eventName} after ${wait}s wait`);
    }, wait * 1000);

    this.debug('debounce', `Debouncing ${eventName} (wait: ${wait}s, max_wait: ${max_wait || 'none'})`);
}
```

✅ **Correct implementation**: Tracks first call time, clears timers, forces execution at max_wait

- ✅ Trailing edge executes last call after interval
- ✅ Handles both leading + trailing combinations
- ✅ Tracks pending data for trailing calls
