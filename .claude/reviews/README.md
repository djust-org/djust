## Usage

### Running a Review

Use the `/review-save` command to review AND save to markdown:

```bash
# Basic review and save
/review-save 42

# Review with additional focus
/review-save 42 Focus on security and performance

# Review external PR
/review-save https://github.com/django/django/pull/123

# Review with custom requirements
/review-save 42 Check for memory leaks and error handling
```

**Command Comparison:**
- `/review 42` - Built-in: Quick review, NOT saved
- `/review-save 42` - This: Same review quality + saved to markdown

### Accessing Reviews

**Latest Review**:
```bash
cat .claude/reviews/pr-42/latest.md
```

**Specific Review**:
```bash
cat .claude/reviews/pr-42/review-2025-01-12-153045.md
```

**All Reviews for a PR**:
```bash
ls -la .claude/reviews/pr-42/
```

Add to `.gitignore`:

```bash

```
