1. Simple Light Navbar

Basic navbar with brand and navigation items.

{{ simple_navbar.render }}
navbar = NavBar( brand={'text': 'SimpleApp', 'url': '/'}, items=[ {'label': 'Home', 'url': '/', 'active': True}, {'label': 'About', 'url': '/about'}, {'label': 'Contact', 'url': '/contact'}, ], variant='light' )

API Reference

Complete parameter reference for the NavBar component.

NavBar( items: List[Dict], # Required: List of nav items brand: Dict = None, # Optional: {'text', 'url', 'logo'} variant: str = 'light', # 'light' or 'dark' sticky: Union[str, bool] = False, # 'top', 'bottom', or False container: str = 'fluid', # 'container' or 'fluid' expand: str = 'lg', # 'sm', 'md', 'lg', 'xl', 'xxl' id: str = None # Auto-generated if not provided ) # Item format: { 'label': 'Link Text', 'url': '/path', 'active': False, # Highlight as active 'disabled': False, # Disable the link 'dropdown': [ # Optional dropdown {'label': 'Item', 'url': '/path'}, {'divider': True}, # Divider line ] }
