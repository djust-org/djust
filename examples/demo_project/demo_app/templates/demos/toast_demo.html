Usage Example

from djust.components.ui import Toast

# Create a success toast
toast = Toast(
    title="Success",
    message="Your changes have been saved!",
    variant="success",
    dismissable=True,
    show_icon=True
)

# Render in template
{{ toast.render|safe }}

Available Options

Parameter Type Default Description
title str "" Toast title (optional)
message str "" Toast message content
variant str "info" Color variant: success, info, warning, danger
dismissable bool True Show close button
show_icon bool True Show icon based on variant
auto_hide bool False Auto-hide after delay
