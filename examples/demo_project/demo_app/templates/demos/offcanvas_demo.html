1. Left Sidebar (Start Placement)

Classic left sidebar for navigation menus.

{{ left_offcanvas.render }}
offcanvas = Offcanvas( title="Navigation Menu", body=""" <nav> <ul class="nav flex-column"> <li class="nav-item"><a class="nav-link" href="/">Home</a></li> <li class="nav-item"><a class="nav-link" href="/about">About</a></li> </ul> </nav> """, placement="start" )

2. Right Sidebar (End Placement)

Right sidebar perfect for filters, settings, or secondary content.

{{ right_offcanvas.render }}
offcanvas = Offcanvas( title="Filters", body="<form>...filter controls...</form>", placement="end" )

3. Top Drawer (Top Placement)

Slide down from the top for notifications or alerts.

{{ top_offcanvas.render }}
offcanvas = Offcanvas( title="Notifications", body="<div class='list-group'>...notifications...</div>", placement="top" )

4. Bottom Drawer (Bottom Placement)

Slide up from bottom for shopping carts or action sheets.

{{ bottom_offcanvas.render }}
offcanvas = Offcanvas( title="Shopping Cart", body="<div>...cart items...</div>", placement="bottom" )

5. Offcanvas Without Backdrop

No backdrop overlay - content remains interactive.

{{ no_backdrop_offcanvas.render }}

Notice: You can still interact with this page when the offcanvas is open!

offcanvas = Offcanvas( title="Settings Panel", body="<form>...settings...</form>", placement="end", backdrop=False # No backdrop overlay )

6. Offcanvas Without Dismiss Button

Force users to use a specific action to close.

{{ no_dismiss_offcanvas.render }}
offcanvas = Offcanvas( title="Important Information", body="<div>...important content...</div>", placement="start", dismissable=False # No close button in header )

API Reference

Complete parameter reference for the Offcanvas component.

Offcanvas( body: str, # Required: Body content (HTML allowed) id: str = None, # Optional: Custom ID (auto-generated) title: str = None, # Optional: Title in header placement: str = "start", # 'start', 'end', 'top', 'bottom' backdrop: bool = True, # Show backdrop overlay dismissable: bool = True, # Show close button in header show: bool = False # Show on page load ) # Trigger offcanvas from HTML: <button data-bs-toggle="offcanvas" data-bs-target="#offcanvasId"> Open </button> # Programmatic control (JavaScript): var offcanvas = new bootstrap.Offcanvas('#offcanvasId') offcanvas.show() offcanvas.hide() offcanvas.toggle()
