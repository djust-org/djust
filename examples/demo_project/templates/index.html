Write Less, Do More

Compare the same interactive counter in djust vs React

djust (Python) 10 lines 
class CounterView(LiveView):
    template_string = """
    <button @click="increment">
        Count: {{ count }}
    </button>
    """

    def mount(self, request):
        self.count = 0

    def increment(self):
        self.count += 1  # Auto-updates!
React (JavaScript) 25+ lines 
import { useState } from 'react';

function Counter() {
  const [count, setCount] = useState(0);

  return (
    <button onClick={() => setCount(count + 1)}>
      Count: {count}
    </button>
  );
}

How It Works

Simple architecture, powerful results

🌐 Browser

~5KB client JavaScript handles events and applies DOM patches via WebSocket

↓

🐍 Django + Channels

Your Python LiveView classes handle events, manage state, and orchestrate updates

↓

🦀 Rust Core

Lightning-fast template rendering (<1ms), VDOM diffing (<100μs), and efficient serialization

The result? Server-side simplicity with client-side responsiveness, all at blazing speed.
