## Mascot Character

The djust mascot is a **cute, cartoonish green snake with sunglasses**:

- Chibi/rounded illustration style
- Dark sunglasses with occasional glint/sparkle
- Green body (`#44B78B`) forming the letter "d"
- Orange gear cog (`#E57324`) at the base where tail meets body
- Friendly, confident expression
- Often paired with **Ferris** (orange Rust crab) as a sidekick

## Usage Guidelines

- Always use the provided files — do not recreate or modify the logo
- Maintain clear space around the logo (minimum: half the icon height)
- Use dark variants on dark backgrounds, light variants on light backgrounds
- Do not stretch, rotate, or alter the logo proportions
- The animated logo is for web use only (hero sections, loading states)
